Somatic Mosaicism across Human Tissues (SMaHT)

Network Associate Member Application

Version 1.2 - Dated August 2025

SMaHT Website: <https://smaht.org/>

SMaHT Data Portal: <https://data.smaht.org/>

Associate members are expected to actively participate in SMaHT Network activities and contribute meaningfully to the Network's outcomes, which are provided below:

Key outcomes of the SMaHT Network

* A catalog of somatic variants in select tissues from human donors

* Innovative sequencing tools and analysis methods that optimize variant detection

* A SMaHT data workbench that seamlessly integrates with current tools to study DNA sequences, like genome browsers, and combines analysis of somatic variation with the current human reference genome

* Fundamentally new ways of understanding the contribution of somatic variation to human biology

To become an associate member of the SMaHT Network, please [complete this form](/download/SMaHT_Associate_Membership_Application_Form_Form_1.2.PDF) and provide the items listed below:

1. A one-page description of proposed research activities and resources (e.g. data production, analyses, software production)

2. A separate one-page document that explains how the activities and resources described above will contribute to the goals of the SMaHT Network.

  - Specifically, how will the applicant contribute to the Network (e.g. attend Working Group or Focus Group Meetings, provide particular expertise to the Network, etc.)?

  - Will the applicant be involved in the Supporting Member's Project? If so, please provide details. If not, please give details about who the applicant plans to work within the SMaHT Network.

3. An updated CV of the applicant
4. A letter of support from a SMaHT Network member

A list of SMaHT awardees can be found on the [SMaHT website](<https://smaht.org/awardees/>).

Only complete applications will be accepted. Please return your complete application to the SMaHT Organizational Center at [smahtsupport@gowustl.onmicrosoft.com](mailto:smahtsupport@gowustl.onmicrosoft.com).
